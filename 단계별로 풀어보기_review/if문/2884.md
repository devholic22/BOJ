# 2884: 알람 시계

### 코드

```python
h, m = map(int, input().split(" "))

if ((0 <= h <= 23) and (0 <= m <= 59)):
    if m >= 45:
        m -= 45
    else:
        minus_m = 45 - m
        if h != 0:
            h -= 1
        else:
            h = 23
        m = 60 - minus_m
    print(f"{h} {m}")
```

### 알게 된 점

- 변수를 출력하고자 할 때, <strong>f-string</strong>을 이용하면 변수를 형식에 맞게 출력할 수 있다.
